# Sample Excel Data Format

Create an Excel file with the following structure:

| Date       | Given To    | Amount | Mode    | Description        | Fund     | Status   |
|------------|-------------|--------|---------|--------------------|----------|----------|
| 2024-01-15 | John Doe    | 150.00 | Cash    | Office supplies    | General  | Paid     |
| 2024-01-16 | Jane Smith  | 75.50  | Card    | Travel expenses    | Travel   | Pending  |
| 2024-01-17 | Bob Johnson | 200.00 | Bank    | Equipment purchase | Equipment| Paid     |
| 2024-01-18 | Alice Brown | 45.25  | Cash    | Food and drinks    | General  | Pending  |

Save this as an .xlsx file and use the Import Excel button to test the functionality.
